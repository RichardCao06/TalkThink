# Sample issue — 复制到 GitHub issue 即可

这是 TalkThink 的第一个种子话题，用来测试两个基础 persona（`zhang-wei` 与 `wang-jianguo`）的对话是否符合 `CLAUDE.md` 的宪法。两人立场在这个话题上直接冲突，是一个适合验证反附和与证据原则是否生效的初始 issue。

---

## 怎么用

1. 在你的 TalkThink GitHub 仓库创建新 issue
2. 用下面的 **Title** 作为标题，**Body** 作为正文
3. 然后给每个想参与的 persona 跑一次 `/discuss <persona-id>`
4. 想让 agent 周期性轮训：`/loop 30m /discuss <persona-id>`

---

## Title

```
现在的劳动法是不是过度保护工人了？
```

## Body

```
民营企业主圈子里流传一种说法：现行劳动法在某些方面过度保护工人，加重企业成本，反而损害就业市场。常被举的例子：

- 合同到期前员工装病拖工伤、拿病假工资
- N+1 赔偿对中小企业是大负担
- 加班时长限制（劳动法第 41 条：每月不超过 36 小时）让企业失去灵活性

另一种说法是相反的：劳动法在执行层面远未到位。大量工人——农民工、外卖骑手、建筑工、餐饮服务员——根本享受不到法律纸面上的保护。出了工伤难以认定，加班限制形同虚设，被欠薪还要自己打官司。

想听不同 persona 的真实看法。请基于自己的经历和价值观回应。引用具体数据请附来源（数字、事件、政策条款都需要可核查）。
```

---

## 为什么从这个话题开始

- `zhang-wei` 在他的 `话题立场` 部分对工伤维权有具体经历和明确态度
- `wang-jianguo` 在他的 `话题立场` 部分明确写了"认为现行劳动法在某些细节上过度保护"的真实立场
- 话题涉及具体可核查的数据（劳动法条款、工伤参保率、农民工社保覆盖率），能测试 `CLAUDE.md` §6 的证据与核查原则
- 直接价值观冲突，能测试 §3 反附和——如果两人最后变成"求同存异"，宪法没工作
